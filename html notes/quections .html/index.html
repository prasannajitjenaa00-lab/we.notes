Interview Questions

1. What is HTML?

HTML stands for Hypertext Markup Language, the language of the Internet. It is the standard text formatting language used for creating and displaying pages on the Internet.HTML documents comprise the elements and the tags that format it for proper page display.

2. What are HTML tags?

HTML (HyperText Markup Language) tags are elements used to structure and format content on a webpage. Tags are enclosed in angle brackets (< >) and usually come in pairs: Opening tag: Closing tag: Self-closing tag: (for elements that don’t require a closing tag, e.g., ,
