Interview Questions

1. What is HTML?

HTML stands for Hypertext Markup Language, the language of the Internet. It is the standard text formatting language used for creating and displaying pages on the Internet.HTML documents comprise the elements and the tags that format it for proper page display.

2. What are HTML tags?

HTML (HyperText Markup Language) tags are elements used to structure and format content on a webpage. Tags are enclosed in angle brackets (< >) and usually come in pairs: Opening tag: <tagname> Closing tag: </tagname> Self-closing tag: <tagname /> (for elements that don’t require a closing tag, e.g., <img>, <br>)

3. What are some common HTML tags?

Some common HTML tags include <h1> to <h6> for headings, <p> for paragraphs,
<a> for links, <img> for images, <ul> and <li> for unordered lists, and <table> for tables.

7. What is the difference between block-level and inline elements?

Block-level elements start on a new line and take up the full width available, while inline elements do not start on a new line and only take up the necessary width to display the content.

9. What is the difference between id and class?

The id attribute is used to uniquely identify a single element on a webpage. It must be unique and cannot be used more than once. The class attribute is used to group multiple elements. It can be used multiple times for styling or applying the same behavior.

10. What is the purpose of the <meta> tag?

The <meta> tag is used to provide metadata about an HTML document, such as the character encoding, viewport settings, or author information.

11. What is the difference between <head> and <body>?

The <head> tag contains metadata about the webpage, such as the title, links to stylesheets, scripts, and other information. It is not visible to users. The <body> tag contains the actual content of the webpage, such as text, images, links, and videos. This content is visible to users.

12. What is the use of the <img> tag’s alt attribute?

The alt attribute in the <img> tag is used to provide alternative text for an image. It is displayed if the image fails to load and is used by screen readers for accessibility. It also helps in SEO.

14.What is the purpose of the <wbr> tag in HTML?
